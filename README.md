This was project was originally inpired by Samson Zhang's video on a neural network in python: https://www.youtube.com/watch?v=w8yWXqWQYmU

Files:
  1. digits.ipynb is my attept at the same problem (github doesn't show the notebook right now but it can be viewed here: https://colab.research.google.com/github/Chirantan-1/Python_NN_testing/blob/main/digits.ipynb )
  2. TheNN.py is a general template to make a neural network.
  3. NNDrawer.py was originally a debugging tool, but I figured it might be useful to test models so I have condensed it into a template.

Examples:
  1. example.py is an example use of TheNN.py, it trains on a model on the mnist database and with the current settings I have got ~97% accuracy.
  2. model.pkl is the model generated by example.py
  3. Guess.py is an example use of NNDrawer.py, it provides an interactive screen to draw on a 28x28 canvas and have the model predict the digit, however the accuracy may not be ~97% as the model was trained on grayscale images but the canvas has 3 colors (white, gray, black).
